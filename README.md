# Rozetka Marketplace Testing Project

Comprehensive testing project for the Rozetka marketplace, featuring both automated and manual testing approaches.

## Project Structure
* **[Automation](./automation)**: End-to-end tests using Playwright and JavaScript with Jira API integration.
* **[Manual](./manual)**: Test documentation, including Test Plans, Bug Reports, SQL scripts, and Postman collections.

## Tech Stack
* **Language**: JavaScript (Node.js)
* **Framework**: Playwright
* **Tools**: SQL, Postman, Jira, Browser DevTools
