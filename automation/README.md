# Automation Testing with Playwright

This section contains automated test scenarios for the Rozetka marketplace platform.

### Key Features:
* **Playwright & JavaScript:** Implementation of a modern framework to ensure reliable and scalable end-to-end tests for the Rozetka marketplace platform.
* **Jira Integration:** Automatic bug report creation via Jira API whenever a test failure is detected during the run.
* **Reporting & Evidence:** Automatic capture of screenshots for failed steps and detailed execution logs to provide clear evidence for debugging.
* **Telegram Bot Integration:** Real-time test execution reports sent instantly to a Telegram channel with summary results (Passed/Failed status for all 51 checklist items).
