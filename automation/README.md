Rozetka Automation Testing Project
Overview
Automated testing system for the Rozetka marketplace, featuring comprehensive test coverage and automated bug reporting.

Key Features
50+ Automated Tests: Comprehensive coverage of user flows using Playwright.
Jira Integration: Automatic bug ticket creation via Jira API upon test failure.
Evidence Collection: Screenshots are automatically attached to Jira issues for every failed test.
Security: Professional implementation of environment variables (.env) to protect credentials.
Tech Stack
Language: JavaScript (Node.js)
Framework: Playwright
Tools: Jira-client, Dotenv
