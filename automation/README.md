# Automation Testing with Playwright

This section contains automated test scenarios for the Rozetka marketplace platform.

### Key Features:
*   Playwright & JavaScript: Implementation of a modern framework to ensure reliable and scalable end-to-end tests.
*   Jira Integration: Automatic bug report creation via Jira API whenever a test failure is detected.
*   Reporting & Evidence: Automatic capture of screenshots and logs to provide clear evidence for debugging.
